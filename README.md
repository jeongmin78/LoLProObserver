# LoLProObserver

#### League of Legends 프로 게이머들의 솔랭 상황을 감지하여 소환사명, 플레이중인 챔피언 및 관전 링크를 slack 메시지를 통해 전달한다.

- python 웹 크롤링을 이용해 op.gg 사이트의 정보 확인 및 가공한다.

- 현재 10초 간격으로 특정 팀의 솔랭 상황을 감지하여 slack 메시지를 보내는 상황으로, 중복 메시지가 계속 전달된다.
